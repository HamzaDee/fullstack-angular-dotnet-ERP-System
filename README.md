# ERP System (Angular + .NET)

## 🚀 Overview
A full-stack ERP system built with Angular and .NET, designed for managing business operations efficiently.

## 🧩 Features
- User & role management
- Dashboard & reports
- Dynamic forms
- Clean modular architecture

## 🛠️ Tech Stack
- Angular
- TypeScript
- .NET Core
- SQL Server

## ▶️ How to run
npm install  
ng serve  

## 📌 Note
This project demonstrates real-world enterprise application structure.
